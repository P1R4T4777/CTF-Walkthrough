# 🧠 CTF Walkthroughs en Español 🇪🇸

Repositorio colaborativo de **walkthroughs de CTFs** enfocados en la **comunidad hispanohablante**, con explicaciones claras, metodología paso a paso y razonamiento técnico detrás de cada acción.

El objetivo es **reducir la brecha de conocimiento** en ciberseguridad ofensiva y facilitar el aprendizaje práctico mediante laboratorios reales.

---

## 🎯 Objetivos del repositorio

- Proveer **walkthroughs técnicos en español**
- Fomentar el aprendizaje estructurado de CTFs
- Crear una base de conocimiento colaborativa para la comunidad

---

## 🧩 Plataformas incluidas

- 🟦 TryHackMe
- 🟩 Hack The Box *(próximamente)*
- 🟥 Otros CTFs (según aportaciones)

---

## 🧪 Metodología utilizada

Los walkthroughs siguen, en general, la siguiente estructura:

1. Reconocimiento
2. Enumeración
3. Explotación
4. Post-explotación
5. Escalada de privilegios
6. Lecciones aprendidas

El enfoque es **práctico y didáctico**, similar a entornos reales de pentesting.

---
